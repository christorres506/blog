What is HTML

HTML is the standard markup language for Web pages, HTML describes the structure of a web page

HTML consists is a series of elements (tags), which you use to wrap different parts of content to make it appear or act in a certain way later on with CSS.

With HTML you will let know the web browser what is a paragraph, header, tittle and so on... You will do this by using tags
