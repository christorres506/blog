What is HTML

HTML is the standard markup language for Web pages, it is the structure of a web page.

With HTML you let know the web browser what is a paragraph, header, and so on... You will do this by using tags.

HTML tags

All tags are wrap with "<>", some examples of tags are

Tags are useful for search engines, which analyze the site classify the content to know how to display the site in the Internet.

They are also useful for accessibility because the screen reader reads each header out as you progress through the content, notifying you what is a header, what is a paragraph, etc. There are two main types of elements.

Block-level element

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

Two commonly used block elements are: <p> and <div>.

Inline element

Inline element does not start on a new line and only takes up as much width as necessary.

Self closing tags

A self-closing tag in HTML is a kind of HTML tag that does not need to be closed manually, which means it does not have a separate closing tag as </tag>. Some few self-closing tags are <input/>, <hr/>, <br/>, <img/>, etc.

Attributes

Attributes can be used to define things such as the height and width of an image, the style of an element (eg. color, size, font).

This how an Anchor tag looks like: <a href="www.google.com">Some text here.</a>

Comments

HTML comments are not displayed in the browser,so this means you only can see this in the code NOT the page! but .Comments can be used to hide content or set of instructions to others developers to let them know what that especific part of the code does.

Everything between the <!-- and the --> will be hidden from the display.

Class Attribute

The class attribute is often used to point to a class name in a style sheet(CSS). It can also be used by a JavaScript to access and manipulate elements with the specific class name(DOM manipulation).The class attribute can be used on any HTML element and you can use multiple classes in one tag.

To define multiple classes, separate the class names with a space

Layout

A page layout the structure of a website, this structure helps the user navigate through web pages easily, you can design web pages using simple HTML tags.

Forms

An HTML form is used to collect user input it contains different kinds of information such as username, password, contact number, email id, etc
