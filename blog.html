What is HTML

HTML is the standard markup language for Web pages, it the structure of a web page

Is a series of elements (tags), which you use to wrap different parts of content to make it look or act in the way you want later on with CSS.(for more advanced styling you also can use a CSS framework like bootstrap!).

With HTML you will let know the web browser what is a paragraph, header, tittle and so on... You will do this by using tags.

HTML tags

All tags are wrap with "<>", some examples of tags are

Tags are useful for search engines, which analyze the site classify the content of the site to know how and when to display the site in the various places on the Internet.

They are also useful for accessibility because the screen reader reads each header out as you progress through the content, notifying you what a heading is, what is a paragraph, etc. There are two main types of elements.

Block-level element

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.

Two commonly used block elements are: <p> and <div>.

Inline element

Inline element does not start on a new line and only takes up as much width as necessary.

Self closing tags

A self-closing tag in HTML is a kind of HTML tag that does not need to be closed manually by its closing tag, which means it does not have a separate closing tag as </tag>. Some few self-closing tags are <input/>, <hr/>, <br/>, <img/>, etc.

Attributes

Attributes can be used to define things such as the height and width of an image, the style of an element (eg. color, size, font) and the language of the document.

This how an Anchor tag looks like: <a href="www.google.com">This is some text.</a>

Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code.Comments can be used to hide content.

Everything between the will be hidden from the display.

Class Attribute

The class attribute is often used to point to a class name in a style sheet. It can also be used by a JavaScript to access and manipulate elements with the specific class name.The class attribute can be used on any HTML element.

To define multiple classes, separate the class names with a space

Layout

A page layout defines the appearance of a website. An HTML layout is a structure that helps the user navigate through web pages easily. It is a way in which you can design web pages using simple HTML tags.

Forms

An HTML form is used to collect user input it contains different kinds of information such as username, password, contact number, email id, etc
