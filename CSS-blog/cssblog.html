What is a style sheet?

A style sheet is a collection of rules that tells a web browser how to display a document written in HTML or XML. A style sheet is written in a text editor and stored in the same folder as the HTML or XML document.

To use a style sheet, the web browser needs to load the style sheet before loading the HTML or XML document.

What is flexbox?

Flexbox is a one-dimensional layout method for arranging items in rows or columns. Items expand to fill additional space or shrink to fit into smaller spaces. This is really useful to save time when building a project.

When used correctly, flexbox can create dynamic and responsive layouts that look great on any device.Use Flexbox to Control spacing and alignment

What is CSS Grid?

Is a two-dimensional grid-based layout system, controlling layout in rows and columns. CSS grid creates complex responsive web design grid layouts more easily and consistently across browsers.
